This file implements functionality for clipboard operations and data transfer.

- **`deleter`**: Implements deleter functionality
- **`unlocker`**: RAII guard ensuring proper resource locking and automatic unlocking on scope exit
- **`clipboard_format`**: Implements clipboard format functionality
- **`state`**: Implements state functionality

### Summary
The `platform.clipboard.cpp` file is essential for clipboard operations and data transfer. It defines 4 class(es) that support Far Manager file manager operations.
